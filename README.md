# Power Glow Sauce Plugin

This is a plugin for Sauce4Zwift that updates an OBS glow source
with the colour associated with the current power output of the
watched athlete.

![Websocket Settings](websocket-settings.png)

- OBS Source: https://obsproject.com/forum/resources/stroke-glow-shadow.1800/
- Sauce4Zwift: https://github.com/SauceLLC/sauce4zwift
